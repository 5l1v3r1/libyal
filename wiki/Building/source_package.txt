== Source package ==
To retrieve the source package go to the [${source_package_url} downloads] page and download the file named:
{{{
${project_name}${project_status}-<version>.tar.gz
}}}

To extract the source package run:
{{{
tar xfv ${project_name}${project_status}-<version>.tar.gz
}}}

This will create the source directory:
{{{
${project_name}-<version>
}}}
